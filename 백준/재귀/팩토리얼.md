재귀 문제 중 가장 기본적인 문제이다. 다음 2가지 방법으로 풀이할 수 있다.

1) 재귀 함수로 푸는 방법
2) 배열을 이용하는 방법

```
n = int(input())

def factorial(n):
    if n == 0 or n == 1:
        return 1
    else:
        f = [0 for _ in range(n+1)]
        f[0] = 1
        f[1] = 1
        for i in range(2,n+1):
            f[i] = i * f[i-1]
            
        return f[n]

print(factorial(n))
```
