## permutations 라이브러리
```
from itertools import permutations
line = list(input())
n = int(line[0])
m = int(line[2])
n_list = permutations([i for i in range(1, n+1)], m)

for n in n_list:
    print(" ".join(map(str, n)))
```


## 재귀 함수
```
line = list(input())
N = int(line[0])
M = int(line[2])

def makeNumber(n):
    if len(n) != M:
        for i in range(1, N+1):
            if i not in n:
                temp = n[::]
                temp.append(i)
                makeNumber(temp)
    else:
        print(" ".join(list(map(str, n))))

for i in range(1, N+1):
    makeNumber([i])
```
